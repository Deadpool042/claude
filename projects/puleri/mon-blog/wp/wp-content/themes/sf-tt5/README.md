# SF TT5 - Notes

## Fix "bloc invalide" sur core/button

Si des pages ont ete creees AVANT la correction des classes de boutons:
- Ouvrir la page dans l'editeur.
- Utiliser "Tenter la recuperation" si propose.
- Sinon, supprimer et reinserer le pattern pour regenerer un markup propre.

Les nouveaux patterns contiennent les classes attendues par WP 6.9
(has-text-color / has-background / wp-element-button).
