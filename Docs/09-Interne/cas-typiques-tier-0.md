# Cas typiques Tier 0 — Projets à capter

> 🔒 **Document interne — ne pas transmettre au client**
>
> Avant Tier 0, ces projets étaient **refusés ou déclassés artificiellement**.
> Avec Tier 0, on peut les **accepter et rentabiliser** (49€/mois).

---

## 🎯 Contexte

Tier 0 ouvre un **nouveau marché d'entrée de gamme** :
- Petites structures (auto-entrepreneurs, petites associations)
- Startups en démarrage (budget limité)
- Projets "test" avant scaling (minimal viable product)
- Demandes urgentes / opportunistes (landing pour campagne)

**Business case** :
- Budget création : 1 200 €
- Maintenance récurrente : 49 €/mois
- Rentabilité : ~25 mois (1 200 ÷ 49)
- Churn faible : site simple = peu de demandes / peu de problèmes

---

## 📋 Cas typiques (exemples concrets)

### **Cas 1 : Auto-entrepreneur / Freelancer**

**Profil** :
- Personne indépendante (coach, consultant, freelancer)
- Besoin : présence basique pour donner crédibilité
- Budget : très réduit (~1 000-1 500 €)
- Contenu : static (CV, offre, contact)

**Site** :
- Landing page + contact
- 3-4 pages (Accueil, Offre, À propos, Contact)
- Pas de vente en ligne
- Stack : Astro statique

**Maintenance** : 10-15 min/mois (certificat SSL, monitoring)

**Exemple** : Coach sportif qui veut un site pro pour trouver des clients

---

### **Cas 2 : Association / ONG petite**

**Profil** :
- Structure à but non lucratif
- Budget d'association (très limité)
- Besoin : présentation + appel à dons / adhésion

**Site** :
- Présentation activités
- Blog simple (actualités)
- Formulaire adhésion / don
- Stack : WordPress minimal

**Maintenance** : 15 min/mois

**Exemple** : Association locale (randonnées, bénévolat) qui veut être visible

---

### **Cas 3 : Landing page d'opportunité**

**Profil** :
- Clientèle existante avec urgence
- Besoin : lancer une page rapidement pour une campagne
- Budget : réduit (contrainte temps)
- Durée : parfois temporaire (campagne électorale, événement, lancement produit)

**Site** :
- One-page ou landing
- Call-to-action unique
- Formulaire inscription / pré-commande
- Stack : Astro ou landing builder léger

**Maintenance** : 10 min/mois (peut être suspendu après campagne)

**Exemple** : Petit événement local qui veut un site pour les inscriptions

---

### **Cas 4 : Blog personnel**

**Profil** :
- Blogueur / écrivain / journaliste indépendant
- Budget : limité
- Besoin : plateforme pour publier sans dépendre des réseaux sociaux

**Site** :
- Blog pur (15-30 articles)
- Aucune e-commerce
- Design épuré
- Stack : Astro ou Ghost minimal

**Maintenance** : 12 min/mois

**Exemple** : Écrivain qui lance son blog sur ses livres

---

### **Cas 5 : Portfolio créatif (v1)**

**Profil** :
- Designer / photographe en démarrage
- Budget : minimal (doit investir le reste dans équipement/marketing)
- Besoin : montrer le portfolio sans lourdeur

**Site** :
- Portfolio galerie (10-20 projets)
- À propos + contact
- Pas de shop (peut venir plus tard)
- Stack : Astro avec galerie native

**Maintenance** : 12 min/mois

**Exemple** : Photographe qui débute et veut montrer son book

---

### **Cas 6 : Vitrine ultra-simple entreprise familiale**

**Profil** :
- PME familiale très petite (restauration, petit commerce, service)
- Propriétaire pas tech
- Budget : limité mais accepte la maintenance

**Site** :
- Infos basiques (horaires, localisation, contact)
- Galerie photos produits / lieu
- Formulaire réservation / info
- Stack : WordPress simple

**Maintenance** : 15 min/mois

**Exemple** : Petit restaurant qui veut afficher ses infos et menus

---

### **Cas 7 : Documentation statique (projet, recherche)**

**Profil** :
- Chercheur / projet open-source / documentation interne publique
- Budget : quasi-nul
- Besoin : rendre du contenu accessible

**Site** :
- Documentation en Markdown
- Navigation simple
- Pas d'interactivité
- Stack : Astro ou Hugo (static generators)

**Maintenance** : 10 min/mois (certificat, mise à jour)

**Exemple** : Chercheur qui veut publier son research paper avec lien

---

## 💡 Comment identifier Tier 0 à la vente

**Indicateurs positifs** :
- Budget client < 1500 € (hint vers Tier 0)
- "On n'a pas de budget IT important"
- "C'est juste pour avoir une présence"
- "On veut quelque chose de simple et rapide"
- "On va peut-être évoluer plus tard"

**Qualifier Tier 0** :
→ Utiliser `grille-qualification-client.md` section Q1

## 📊 Impact business Tier 0

### Avantages agence

✅ **Rentabilité par volume** : 10-15 projets Tier 0 = ~3000 € récurrent
✅ **Churn bas** : site simple = peu de problèmes = retention haute
✅ **Effet viral** : petit client devient grosse PME en 5 ans
✅ **Portfolio** : diversifie le portefeuille (pas juste PME/ETI)

### Risques à éviter

⚠️ **Creep scope** : client demande blog + module + multi-langue → requalification obligatoire
⚠️ **Support non borné** : établir clairement les limites dans contrat
⚠️ **Churn par upgrade** : client passe en Tier 1+ → bon pour récurrent, mauvais si perte de marge
⚠️ **Mauvaise qualification** : confondre Tier 0 avec Tier 1 → problème de rentabilité

## 🎯 Action commerciale

**Pour chaque prospect Tier 0** :

1. ✅ Qualifier avec Q1 de `grille-qualification-client.md`
2. ✅ Présenter avec `guide-tier-0-1.md` (commercial)
3. ✅ Fixer attentes claires (maintenance obligatoire, scope clair)
4. ✅ Contrat standard Tier 0 (minimal = pas de scope creep)
5. ✅ Planche bonne expérience (on veut qu'il upgrade vers Tier 1+ ou recommande)

---

## 📋 Checklist avant acceptation Tier 0

- [ ] Client accepte 1 200 € création + 49 €/mois maintenance
- [ ] Budget pas de e-commerce avancé, multi-langue, ou modules complexes
- [ ] Client a le contenu prêt (ou très minimal)
- [ ] Délai acceptable (2-3 semaines pour Tier 0)
- [ ] Hébergement compatible avec base technique agence
- [ ] Client comprend la maintenance obligatoire

---

## 📌 Règle fondamentale

> **Tier 0 n'est pas "un site au rabais".**
> C'est une offre **légitime et professionnelle** pour les petits budgets.
> Bien exécuté, c'est une porte d'entrée vers des clients qui grandissent.
