| **Part du marche** | Petit volume | 70-80 % | 15-25 % | 5-10 % | Ponctuel |

### Tier 2 (base 3 500 EUR + modules)

| Element | Montant |
|---------|--------:|
| Base Tier 2 | 3 500 EUR |
| Modules | Selon modules activés |
| **Total** | **Déterministe** |

### Tier 3 (base 5 600 EUR + modules métier)

| Element | Montant |
|---------|--------:|
| Base Tier 3 | 5 600 EUR |
| Modules métier | Selon modules activés |
| **Total** | **Déterministe** |

### Tier 4

- Prix de base : sur devis (app / plateforme, headless avancé)
- La base varie selon la famille technique et la stack (voir tableau commercial)

> Prix total déterministe : prix de base (selon la stack) + modules + mise en production = total.

- Les modules sont factures a part
- La maintenance est hors creation
- Toute demande hors cadre = requalification
